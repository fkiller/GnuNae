Overview

GnuNae is an AI-powered browser that integrates OpenAI's Codex CLI for intelligent web automation. It provides a sidebar interface for AI-assisted browsing, task automation, and page analysis.

‚ö†Ô∏è Test Account Requirements

Testing AI features requires a ChatGPT Pro or Plus subscription from OpenAI. Free ChatGPT accounts cannot use Codex features. You can upgrade at chat.openai.com.

Authentication Steps

  1. Launch GnuNae
  2. Click "Sign in to OpenAI" in the sidebar
  3. Complete the OAuth flow with your OpenAI credentials
  4. After authentication, the sidebar will show "Ready"

Important Dependencies

  • Internet Connection: Required for AI features (OpenAI API)
  • Node.js Runtime: Bundled with the Windows package in resources/runtime
  • Codex CLI: Bundled with the Windows package in resources/codex

Key Features to Test

  • Browser Navigation: Use the address bar to navigate to websites
  • AI Chat: Type prompts in the sidebar (e.g., "Summarize this page")
  • Task Manager: Save and schedule automated tasks (Settings ‚Üí Task Manager)
  • Personal Data Store: The app can remember user preferences locally
  • Multi-Tab Support: Ctrl+T opens new tabs; Ctrl+W closes tabs
  • Multi-Window: Ctrl+N opens a new window

üí° Virtual Mode (Optional)

Virtual Mode requires Docker Desktop installed. If Docker is not available, the app operates in Native Mode, which is fully functional.

No Background Services

GnuNae does not install background services or drivers. The app runs entirely within its own process space.

‚úÖ No Admin Rights Required

Standard user permissions are sufficient for all functionality.

Data Storage Locations

  • Settings: %APPDATA%\GnuNae\
  • Personal Data Store: %USERPROFILE%\.gnunae\datastore.json
  • OpenAI credentials: Stored via Codex CLI's secure storage

Known Limitations

  • AI features require active internet connection
  • Full AI functionality requires ChatGPT Pro/Plus subscription
  • Virtual Mode requires Docker Desktop (optional)
